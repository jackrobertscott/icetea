## Installation

Using npm:

```shell
npm i --save lumbridge
```

Using yarn:

```shell
yarn add lumbridge
```

Then import the helper classes where needed.

```js
import { Router } from 'lumbridge';

const router = Router.create({
  // code...
});
```
